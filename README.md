## Installer

-   `git clone https://github.com/mirzapurnandi/jendela360.git`
-   `cd jendela360`
-   `composer install`
-   `cp .env-example .env`
-   `php artisan key:generate`
-   `php artisan migrate`
-   `php artisan db:seed`
